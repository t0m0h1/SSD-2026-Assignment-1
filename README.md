# Patient Record Management System

## Overview
A secure Flask web application for managing patient records with authentication and role-based access control.

## Features
- User login and registration   
- Patient record management  
- CRUD operations
 

## Tech Stack
- Python (Flask)  
- SQLite  
- HTML/CSS  

## Run the App
in bash:

pip install -r requirements.txt

python app.py

(use a virtual environment if needed)

(use specific pip relevant to the python version, e.g. pip3.12 install ...)
